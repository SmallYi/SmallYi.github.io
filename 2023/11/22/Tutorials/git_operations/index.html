【教程】Git 操作命令

Author: 初夏
Date: 十一月 22, 2023  10:11:12
Category: 教程

git 查看操作

  • git 查看操作记录
  • 使用图形化界面工具 gitk 来更直观地查看操作记录

  • 通过 git log 命令可以查看提交历史
    • git log path/to/file 可以列出该文件的所有提交记录
    • git log -n 命令，其中 n 是想要查看的记录数量
    • 使用 git log –merges 命令来查看分支的合并历史
  • 使用 git show 命令可以查看单个提交的详细信息
  • 通过 git diff 命令可以比较提交之间的差异
    • git diff commit1 commit2 可以显示 commit1 和 commit2 之间的差异
  • 使用 git blame 命令可以逐行查看文件的修改历史
